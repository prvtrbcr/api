# simple-api

simple-api adalah source code API menggunakan expressJS, docs swagger UI

## Instalasi

1. **Clone Repositori:**
   ```bash
   git clone https://github.com/ArifzynXD/simple-api 
   cd simple-api 
   ```
2. **Instal Dependensi:**
   ```bash
   npm install or yarn install
   ```
3. **start the server:**
   ```bash
   node . or npm start
   ```
